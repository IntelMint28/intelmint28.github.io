ICOMMITEDTAXEVASION-PARSER: convert images to a messy, uncompressed XML format

icommitedtaxevasion-parser is an open source tool developed by Eunice28 (ME, this is an alias) as a joke python program to convert .PNG files to a brand new .ICOMMITEDTAXEVASION image format. These files, are uncompressed, raw PNG data saved in an XML-based, uncompressed pixels. Despite having no compression and massive file sizes (1000x1000 images can use almost 1 million lines and over 60MB), you can move this media without people even knowing you are moving an image. The .icommitedtaxevasion format came to light when i was researching how to edit .BRLYT and .BRLAN files. The name is a joke on how some people make joke confesses that they performed tax evasion. Since there is no compatible applications with .icommitedtaxevasion image files, you can "de-parse" these messy files to PNG files by switching the script mode to "DEPARSE". .icommitedtaxevasion files also have image headers such as PNG or JPEG files despite being a chaos inside the file: Image Resolution, Image Color Mode, Original Image Format, DPI and even Comments. In order to compress your PNG to a .icomittedtaxevasion image, you need at least 12GB of RAM to compress medium-quality images, due to its unefficiency. A sequel to this format is planned on the future by the name of ".thefbiistrackingmedown" that compresses video files in the same messy way as .icommitedtaxevasion (with compression since, using raw data too can make a 5 seconds video 1GB file size, which is unefficient, but I dont want it to be way too unefficient)
